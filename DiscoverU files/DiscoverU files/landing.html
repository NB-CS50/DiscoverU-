Quick Tips

Power-user habits to build from day one

⚡
Check Daily
Log in every morning. Lecturers often send time-sensitive updates about classes and assessments.
📂
Folder Per Module
Create a folder for each course at the start of semester. Use rules to auto-route emails there.
✅
Flag Deadlines
Whenever an email mentions a deadline, immediately flag it and set a due date reminder.
🖊️
Signature First
Set up your professional signature before sending your very first email — it signals seriousness.
🔵
Use Categories
Colour-code by urgency: red for urgent, blue for academic, green for administrative emails.
🤖
Summarise Long Threads
For group project threads that spiral, use Copilot AI Summary to get the gist in one click.
